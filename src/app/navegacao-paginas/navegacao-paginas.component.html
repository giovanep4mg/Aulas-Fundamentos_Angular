Navegação entre páginas


Como fazer a navegação entre as páginas.

Dentro do app.component.html acima do "router-outlet",
coloque as tags:

nav dentro da nav, coloque,
ul será coluna de listas, dentro dela ,
li seria a linha, dentro dela,
a routerLink="primeira-pagina"> "Primeira página" /a
/li fecha o "li", e adiciona outra "li",
li seria a linha, dentro dela,
a routerLink="segunda-pagina"> "Segunda página" /a
routerlink é onde você coloca o link para a página que deseja ir.
/li fecha o "li",
Veja como fica no código:


Agora temos que cria essa "Segunda página", para isso:
Eu criei um novo componente, chamado "segunda-pagina",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.


Depois de criado vai no template dele segunda-pagina.component.html,
coloque um "h2" com um titulo = "Segunda página", feche o "/h2".

Agora para aparecer tem que criar uma rota para ele, vai no app-routing.module.ts ,
e acrescenta mais uma rota * path:'segundo-componente', component:SegundoComponentComponent * ,

Abra a página e veja se está funcionando os links das páginas, a navegação.
Perceba que o menu de navegação não se mexe, mesmo com muda de página,
isso acontece por que o "nav" está acima do "router-outlet" as páginas serão carregadas no "router-outlet".


Criar um botão, uma função que leve para outra página

Será criado dentro da primeira página, para isso:
Vai no componente primeira-pagina.component.html, e vamos adicionar essas tags:
button type="button" (click)="moverParaSegundaPagina()" "Ir para a segunda página",
fecha o "/button".

Agora temos que criar esse método, vai na classe primeira-pagina.component.ts,
dentro do "constructor" adicione o serviço de rotas, colocando dentro dele como parâmetro :
private router:Router .

Agora podemos criar o método:
moverParaSegundaPagina() ,abre "chaves"
this.router.navigate(["segunda-pagina"]); , fecha as "chaves".

navigate([ " *rota* " ]), dentro dele você coloca para onde quer ir.
