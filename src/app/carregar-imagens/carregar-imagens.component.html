Carregar imagem


Primeiro:
Eu criei um novo componente, chamado "carregar-imagens",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Como carregar uma imagem?
Você deve colocar a imagem na pasta "assets".
Depois no componente carregar-imagens.component.component.ts, na classe e adiciona,
a imagem dentro de uma propriedade, exemplo urlimagem = './assets/nego1.jpg';.
Criei a propriedade "urlimagem" e dentro dela coloquei onde está a foto,
assim quando usar a propriedade "urlimagem", vai chamar a imagem.

Para usar essa imagem, no template, ou seja, na página você deve chamar usando a tag img,
usando esse comando : < img [src] = "urlimagem" width = "300" height = "300" > ,
passei a propriedade, e também já adicionei um tamanho para a imagem, width = largura, heigth = altura.

Você também pode "chamar" a imagem direto sem precisar cria a propriedade, basta colocar o caminho da imagem,
A imagem deve estar na pasta assets, ai quando for chama-la passe o caminho, assim:
cria a tag "img" < img src = "./assets/nego1.jpg" width="300" height="300" > ,
também já coloquei um tamanho que ela vai ficar.
já será exibida na página.
