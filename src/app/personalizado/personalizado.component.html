Componente Personalizado


Eu criei um novo componente, chamado "personalizado",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Atenção: já coloque o componente criado para aparecer no template,
para isso vai no componente app.component.html, adiciona o componente criado,
app-personalizado, verifique se está aparecendo no template e pronto.

Agora que já foi criado o componente, vai na classe personalizado.comonent.ts,
adiciona um "decorador de input" @Input(), e ai você defini qual propriedade quer usar,
coloquei nome = "" ; vai iniciar vazio, adiciona outra propriedade,
adiciona um "decorador de input" @Input(), dessa vez a propriedade será sobrenome = ""; ,
que também se iniciará vazia.
