two way data binding


Eu criei um novo componente, chamado "two-way-data-binding",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Atenção: já coloque o componente criado para aparecer no template,
para isso vai no componente app.component.html, adiciona o componente criado,
app-two-way-data-binding, verifique se está aparecendo no template e pronto.

É uma troca de dados, entre o componente e o template,

Depois de criar o componente, vai no two-way-data-binding.component.ts,
dentro dele, na classe adicione a propriedade nome = "" ; .

Agora vai no componente two-way-data-binding.component.html,
adicione uma tag "h3", só para colocar um titulo e já pode fechar ela "/h3",
abaixo dela um "p", dentro abre "chaves duplas", coloque nome ,
feche as "chaves duplas", e o "p".

Mas como a propriedade "nome" está vazia, não irá aparecer nada no template,
agora vamos pegar o dado do template, enviar para a componente e vice versa.

No template mesmo, abaixo do "p", adicione:
"label" com for = "nome" , e dentro dela digite:
"Digite seu nome:", fecha a "label", abaixo dela coloque um,
"input" do tipo texto = type = "text".
Para facilitar e fazer mais sentido, coloque a tag "p",
abaixo do "input", e acrescenta um texto para marca,
"O nome digitado é: ",
