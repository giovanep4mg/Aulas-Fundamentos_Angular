Primeiro:
Eu criei um novo componente, chamado "card",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Para reutilizar códigos, quando precisa adicionar componentes dentro de outros componentes.
Você criou vários componentes, como foi feito até agora, ai podemos separar eles, colocando um titulo, uma borda,
É isso que iremos fazer mas sem precisar fazer para todos, para não ter muito trabalho.

Vamos criar como se fosse uma capa, para todos os componentes, onde só adiciona um titulo e uma cor,
os estilos serão adicionados somente uma vez e depois todos os componentes irão conter eles.

Vamos começar!
Depois de criado o novo componente card, vai na classe dele card.component.ts,
adiciona as propriedades que ele vai receber:

@Input() titulo = " " ; => que vai receber a propriedade "titulo", e inicializa vazia.
@Input() cor = " " ; => que vai receber a propriedade "cor", e inicializa vazia.
