Parâmetro da rota

Observe essa imagem:
Temos uma "url" que contém:
http: => que é um esquema pode ser "http ou https".
//localHost => que é o "domínio".
: 4200 => que é a porta.
/produtos => que é a rota.
/1 => que é o parâmetro da rota.

Serve para obter informações, detalhes de um determinado produto ou recurso.


Agora vamos criar um novo componente, vai no pront de comando e digita:
ng g c pagina-com-parametros, dá enter.

Depois de criado o componente vai no app-routing.module.ts e adiciona:
path: 'pagina-com-parametros/:id', component: PaginaComParametros

Esse "/:id" é o parâmetro que terei que acessar através da classe,
quando precisar obter essa informação.

Agora no template pagina-com-parametros.component.html, adicione essa tag:
"h2" dentro dela esse texto ='Parâmetro da rota', abre "chaves duplas" dentro coloque id.
Vai dá um erro, por que esse id não existe, ainda não foi criado.

Para criar o id, vai na classe pagina-com-parametros.component.ts,
no "constructor" passe como parâmetro o " private route : ActivatedRoute ",
através dele podemos trabalhar com 'rotas'.

Depois no "ngOnInit()", vamos dizer que queremos pegar um "id",

this.route.paramMap.subscribe((params) =>
this.id = Number(params.get('id'));
para criar esse "id", dentro da classe adicione a propriedade:
id: number | null = null;

Como obter os parâmetros de consulta

Vai no template pagina-com-parametros.component.html e adiciona mais essas tags:

h2 *ngIf="nome">Query param de nome:"chaves duplas" dentro = 'nome', /h2
h2 *ngIf="idade">Query param de idade:"chaves duplas" dentro = 'idade', /h2

*ngIf serve para, que se estiver "nome ou idade " vai aparecer no template.
