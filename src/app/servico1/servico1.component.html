As funçoes são a mesma, e para tratar essas funcões vou cria um "serviço".

No pront de comando, crie um service:
ng g service logger, clique em enter.

Depois de criado vai no logger.service.ts
dentro dele pode ser implementada todas as funcionalidades,
decorator = @Injectable que é para poder injetar esse serviço em nossos componentes.

Dentro da classe vamos criar uma função:
logar(mensagem:string) que vai receber uma mensagem.
console.log(mensagem); e vai mostrar no console.

Já vamos criar tambem uma lista, "vetor" de mensagens.
mensagens: String[] = [];

No método "logar", adicione esse código, para que toda vez que logar,
vai adicionar a mensagem dentro do "vetor" da lista.
this.mensagens.push(mensagem); Agora veja como fica o código:


Depois vamos adicionar outro método "exibirTodosOsLogs()",
que vai mostrar no console todas as mensagens:
exibeTodosOsLogs() abre chaves,
console.log(this.mensagens);, fecha as chaves.

Agora vamos ultilizar esse serviço com esses métodos nos componentes

Vai no serviço1, componente servico1.component.ts,
vamos injetar nosso "serviço" dentro desse componete.

Tem que criar o constructor e injeta dentro dele,
constructor(private logger: LoggerService).
Agora já posso usar os métodos criado no "serviço".

No método "adicionarNome", vou modificar o código:
this.logger.logar(`O nome $ * this.nome * foi adicionado`);
Agora veja como fica o código:

Faça o mesmo com o servico2, só adicionar o constructor,
passar os parâmetros "private logger: LoggerService" e modificar o método,
this.logger.logar(`O nome $ * this.nome * foi adicionado`);
E para finalizar vamos adicionar o método "exibir todos os logs",
que vai mostrar os dois logs "nome do produto e descrição dele".

Para isso, no componente servico2.component.html, adicione:
button type="button" (click)="this.logger.exibeTodosOsLogs()"
com o texto = "Exibe todos os Logs", e feche o "/button".
