Usar o Pipe "|" para fazer formatações no texto


Eu criei um novo componente, chamado "piper",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

No componente piper.component.ts, que é classe dele, adicionei a propriedade,
nome = "giovani";,
dataNascimento = "11/08/1983" ;,
dinheiro = 200000;,
E ao chamar essas propriedades no template adiciono o piper, para fazer as formatações.

Você coloca o texto dentro das chaves duplas e o pipe, como está a imagem abaixo:

Do lado do piper você passa a formatação que você quer que seja feita,
vamos usar o uppercase, deixando todo texto com letras Maiúsculas.
Como fica na página:

Nome: {{ nome | uppercase }}


Você pode usar para fazer várias coisas,
formatando a data de nascimento, para como nós usamos.

Converter número em dinheiro:
Dentro do componente piper.componet.ts,
adicionei a propriedade "dinheiro = 200000",
e na hora de chamar no piper.component.html,
dentro das "chaves duplas", adicionei a propriedade e
usei o "pipe" currency : "BRL",
que já fez a formatação e adicinou o "R$, a vírgula e o ponto".
Ficam assim, no template:

{{ dinheiro |currency: "BRL" }}


Converter a data "11/08/1983" em TEXTO :
Dentro do componente piper.componet.ts,
adicionei a propriedade "dataNascimento = "11/08/1983",
e na hora de chamar no piper.component.html,
dentro das "chaves duplas", adicionei a propriedade e
usei o "pipe" date | uppercase,
que já fez a formatação da data em texto.
Ficam assim, no template:

{{ dataNascimento | date | uppercase }}
