Criando o "pipe" customizado

Eu criei um novo componente, chamado "pipe-customizado",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g pipe*nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g " significa = ng generate .

Já vimos como usar o pipe, e agora vamos criar um pipe-customizado,
vai no pront de comando, verifique se está dentro da pasta do seu projeto.

Adicione esse comando ng g pipe multiplica dá enter,
já vai ser declarado e importado dentro do app.module.ts.

Ele não vai ficar dentro de nenhuma pasta, fica perto da pasta "assets",
dentro do componente multiplica.pipe.ts, você vai ver:
"decorator" e esse nome é o que temos que usar quando for usar o pipe.

E também a classe "implementa" o PipeTransform,como está na imagem:

Logo embaixo, já tem um método transform,
que tem que passar alguns parametros:

value: que é o valor.
unknowno: que é do tipo, "number, string,...".
args: que é os argumentos.
unknowno: que é do tipo, "number, string,...".
unknowno: que fica sem nada mesmo

return valor * "argumento" ;


Agora que foi criado, vamos testar ele dentro do template,
vou cria um "h2", só para colocar um titulo, fecho o "/h2",

depois um "p" um paragráfo:
dentro dele "duas chaves duplas",
dentro dela 2 | multiplica: 4 ,
fecho as "duas chaves duplas" e fecho o "p".
