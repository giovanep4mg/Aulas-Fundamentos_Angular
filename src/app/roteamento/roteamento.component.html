Roteamento

Vamos criar um novo projeto, com o nome "roteamento", para isso:
No pront de comando do "VScode" digite esses comandos:
ng new *nome do seu projeto* .
Vai pedir para você confirma se quer ou não instalar o "Angular routing" diz que não,
por que vamos ver como adiciona ele passso a passo.
Escolha o "CSS" , e espere por que vai demorar um pouquinho.

Agora depois de criado o projeto, vamos instalar a biblioteca que vamos usar.
no pront de comando, verifique se você está na pasta do projeto, tem que está dentro dela;
para "Entrar" na pasta use o cd *nome da pasta ,
para "Sair" da pasta use o cd..

Na pasta do projeto, para instalar a biblioteca adicione o comando:
npm i @angular/router => para trabalhar com roteamento de páginas.

Agora tem que criar um módulo responsável por fazer esse roteamento,
no pront de comando ultiliza esses comandos:
ng g module app-routing --flat --module app, dá enter.

--flat => usa para ele já ficar na pasta "app" .

--module => já vai acrescenta ele dentro do "app.module.ts" .


Vamos começar a trabalhar esse módulo de roteamento, no app.routing.module.ts .
No "@ngModule" dentro do "imports" adiciona o RouterModule.forRoot(routes) ,

para dizer que vai importa "rotas", e agora vamos criar essas rotas:

Acima do "@ngModule", você coloca esses comandos:
const routes: Routes = [ *aqui dentro passa a rota* ], já vamos adicionar a primeira rota
* path: 'primeira-pagina', component: PrimeiraPaginaComponent *, no lugar dos asteriscos são "chaves".

path é o endereço que digitamos na barra do navegador.
component é qual componente vai inicializar.

Vai dá erro por que precisamos criar esse "PrimeiraPaginaComponente".
Para criar esse componente, vai no pront de comando e coloque esses códigos:
ng g c primeira-pagina , dá enter.

Agora vamos dá uma atualizada no "PrimeiroPaginaComonente" para o angular imorta ele.

Agora vai no template dele primeira-pagina.component.html, e adiciona as tags:
h2 coloque o titulo "Primeira Página", feche o /h2

Volte agora no app.rounting.ts , para exportar e pode ser usado no "app.module".
exports: [RouterModule],

Para finalizar vai no componente app.component.html, apagar o que estiver lá dentro,
e adicona o router-outlet , para conseguir ver na página.

Para rodar o projeto, vai no pront de comando e digita:
ng serve --open , que vai carregar o projeto e já vai abrir no navegador.

Pronto finalizamos nossa primeira aula.
