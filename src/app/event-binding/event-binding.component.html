Event Binding - Vinculação de Eventos


Eu criei um novo componente, chamado "event-binding",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Atenção: já coloque o componente criado para aparecer no template,
para isso vai no componente app.component.html, adiciona o componente criado,
app-event-binding, verifique se está aparecendo no template e vamos começar...

É quando o usuário vai clicar num botão ou link e aciona um evento,
vai ter uma ação, esse evento pode ser métodos, criados na classe.

Vamos criar alguns exemplos, na classe do nosso componente,
vai no event-binding.component.ts e adiciona essas propriedades:

nome = "Giovani";
dataNascimento = "12/02/1989";
cidade = "Passa-Quatro, MG"
idade = "38";
