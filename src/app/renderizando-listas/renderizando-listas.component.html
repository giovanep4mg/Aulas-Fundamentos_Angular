Renderizando Listas

Eu criei um novo componente, chamado "renderizando-listas",
Para criar, vai no pront de comando do "VScode", verifique se você está dentro da pasta do seu projeto,
coloque esse comando ng g c *nome do componente que deseja criar*, dá "enter" e ele vai criar para você.
Esse comando "ng g c" significa = ng generate component.

Atenção: já coloque o componente criado para aparecer no template,
para isso vai no componente app.component.html, adiciona o componente criado,
app-renderizando-listas, verifique se está aparecendo no template e vamos começar...

Nessa aula vamos ver como renderizar uma lista, usar diretivas, mostrar ou não elementos dessa lista.

Agora para mostrar essa lista, vai no componente renderizando-listas.component.html,
dentro dele adicione essas tags:
"h2" só para criar um titulo e fecha o "/h2",
"ul" para criar uma lista, dentro dele
"li" para cada celular um 'li', mas para isso usamos o "*ngFor
dentro dessa "li", vai usar um diretiva *ngFor="let Celular of celulares,

crie uma tag "strong", dentro dela "chaves duplas", dentro delas:
Celular.nome, fecha a tag "/strong", abaixo crie um ,
"p" dentro dele "chaves duplas" dentro delas Celular.descricao, fecha o "/p"

crie um "span", chaves duplas e dentro delas adicione:
Celular.esgotado ? 'esgotado' : 'disponivel', está fazendo uma pergunta para a lista,
o celular está esgotado?, se estiver "true" lá na classe, vai mostrar "esgotado",
se estiver "false" na classe a mensagem será "disponivel".
